Tax API

Edgewater Tax Engine — /api/health

Endpoints

GET  /api/health              — public
GET  /api/forms               — list 19 supported forms
GET  /api/tax-tables/:year    — IRS brackets 2020-2025
GET  /api/field-map/:form/:year — Textract-verified field maps

POST /api/compute/1120        — C-Corp return
POST /api/compute/1120s       — S-Corp return
POST /api/compute/1040        — Individual return
POST /api/compute/cascade     — S-Corp → K-1 → 1040

POST /api/compute/ordinary-tax
POST /api/compute/qbi
POST /api/compute/niit
POST /api/compute/standard-deduction

POST /api/fill/:form/:year    — fill PDF from field IDs
POST /api/label/:form/:year   — label fields for verification
POST /api/verify              — Textract round-trip verify

POST /auth/signup             — create account
POST /auth/signin             — get session
GET  /auth/me                 — profile + API keys
POST /auth/api-keys           — create key
DELETE /auth/api-keys/:id     — revoke key

POST /api/scenarios           — create scenario
GET  /api/scenarios           — list scenarios
POST /api/scenarios/:id/compute  — run engine
POST /api/scenarios/:id/analyze  — Gemini AI analysis
POST /api/scenarios/compare      — compare scenarios

Auth
